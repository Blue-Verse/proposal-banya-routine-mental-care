핵심 요구사항 분석

  • 소셜 간편 로그인 — 카카오, 네이버, 애플 계정 연동 (Firebase Auth 활용)
  • 초기 진단 (온보딩) — 3~5개 객관식 문항으로 '기운 DNA(성향)' 파악
  • 자체 캐릭터 UI 연동 — 라토, 호담 캐릭터를 튜토리얼 안내 및 미션 달성 리액션으로 활용
  • 기세 예측 캘린더 — 초기 진단 데이터 + 날씨 API + ChatGPT API 결합으로 '오늘의 기운 상태' 자동 생성
  • AI 맞춤형 멘탈 코칭 (RAG 필수) — 자체 텍스트 데이터를 벡터 DB화하여 검색 증강 생성 방식 응답
  • 데일리 푸시 알림 — 아침 미션 알림 + 미수행자 저녁 리마인드 + 월간 리포트
  • 정기 구독 결제 — 포트원 연동 월 4,900원, 무료/유료 권한 분리
  • 커머스 연동 — 스마트스토어 아웃링크 버튼
  • 관리자 페이지 — 사용자 관리, 콘텐츠 관리, 데이터 분석

제안 포인트

본 프로젝트는 캘린더 앱 + AI(RAG) 코칭 + 게이미피케이션 미션 + 구독 결제가 결합된 복합 서비스입니다. 아래와 같은 이유로 효율적이고 높은 품질의 MVP를 납품할 수 있습니다:

  • 캘린더 앱 직접 개발 경험 — Flutter 기반 소셜 캘린더 앱(Calendar Share)을 기획부터 배포까지 완수. 캘린더 라이브러리 커스텀, 이벤트 관리, 푸시 알림 7종 구현 경험 보유
  • 게이미피케이션 + 캐릭터 시스템 경험 — DayStarter 앱에서 Spine 2D 캐릭터 시스템, 데일리 미션, 루트박스 이코노미, 행동 변화 유도 메커니즘을 구현. '반야루틴'의 캐릭터 연동 + 미션 시스템과 직접적 유사성
  • AI/RAG 기술 역량 — AI 에이전트 프레임워크를 직접 설계·구축하며 LLM API 연동, 벡터 검색, 프롬프트 엔지니어링에 깊은 경험 보유. RAG 파이프라인(임베딩→벡터 검색→응답 생성) 구축 가능
  • 결제 + 구독 시스템 경험 — 복수 프로젝트에서 결제 연동(포트원, PG사) 및 유료/무료 권한 분리 구현 경험
  • 3개월 내 Flutter 앱 풀패키지 딜리버리 — Fortune App(운세 앱)을 3개월 내 기획부터 스토어 출시까지 완수한 실적. 본 프로젝트도 동일한 일정 내 MVP 완성 가능

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

3040 세대의 감정 자본을 관리하고 매일의 기세를 예측해주는 모바일 앱 '반야루틴'의 MVP를 구축합니다. 생체 심리 데이터와 생성형 AI를 결합하여, 유저에게 매일 맞춤형 오프라인 행동 미션을 처방하고 일상의 기세를 관리하는 서비스입니다.

  • 타겟 사용자: 3040 세대 (30~40대) 직장인 및 자영업자
  • 핵심 가치: 매일의 감정/기운 상태를 AI가 예측하고, 맞춤형 행동 미션으로 기세 관리
  • 비즈니스 모델: 월 4,900원 정기 구독 (프리미엄) + 굿즈 커머스
  • 플랫폼: AOS/iOS 크로스플랫폼 (Flutter)

3. 기능 요구사항

3.1 소셜 간편 로그인 및 초기 진단 (온보딩)

  • [필수] 카카오, 네이버, 애플 계정 연동 간편 로그인
  • [필수] 신규 유저 최초 로그인 시 3~5개 객관식 문항으로 '기운 DNA(성향)' 파악
  • [필수] 진단 결과를 유저 프로필에 저장하여 AI 코칭 및 기세 예측에 활용

3.2 자체 캐릭터 UI 연동

  • [필수] 클라이언트 제공 캐릭터(라토, 호담) 이미지 소스 앱 내 통합
  • [필수] 앱 최초 실행 시 캐릭터 기반 튜토리얼 안내
  • [필수] 캘린더 미션 달성 시 캐릭터 완료 스탬프/리액션 표시

3.3 기세 예측 캘린더

  • [필수] 오픈소스 캘린더 라이브러리를 서비스 톤앤매너에 맞게 커스텀
  • [필수] [초기 진단 데이터] + [오늘의 날씨 API] + [ChatGPT API] 결합
  • [필수] '오늘의 기운 상태' 텍스트를 캘린더 상단에 자동 생성 및 노출
  • [필수] 일별 미션 표시 및 달성 여부 추적

3.4 AI 맞춤형 멘탈 코칭 (RAG)

  • [필수] 유저가 고민을 텍스트로 입력하면 AI가 맞춤형 솔루션 답변
  • [필수] 자체 텍스트 데이터(스레드, 블로그 칼럼 등)를 벡터 DB화
  • [필수] RAG(검색 증강 생성) 방식으로 응답 로직 구축
  • [선택] 대화 이력 저장 및 맥락 유지
